# scholarship-form-project

Dự án xây dựng **biểu mẫu xin học bổng trực tuyến** cho sinh viên Trường Đại học Xây Dựng Hà Nội, sử dụng HTML, CSS và JavaScript. Biểu mẫu cho phép sinh viên nhập thông tin cá nhân, học tập, thành tích, lý do xin học bổng và gửi đơn trực tiếp trên trình duyệt.

---

## Cấu trúc thư mục

```text
scholarship-form-project/
├── index.html          # Trang chính: biểu mẫu xin học bổng
├── style.css           # Định dạng giao diện
├── script.js           # Xử lý logic, validate dữ liệu, lưu localStorage
├── images/
│   └── pexels-pixabay-267885.jpg  # Ảnh banner đầu form
└── README.md           # Tài liệu giới thiệu ngắn
```
