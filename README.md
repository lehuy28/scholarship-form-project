# scholarship-form-project

Dự án xây dựng **biểu mẫu xét học bổng trực tuyến** cho sinh viên Trường Đại học Xây Dựng Hà Nội, sử dụng HTML, CSS và JavaScript. Biểu mẫu cho phép sinh viên nhập thông tin cá nhân, học tập, thành tích, lý do xin học bổng và gửi đơn trực tiếp trên trình duyệt.
